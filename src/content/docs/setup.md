# Pierwsze kroki (Setup)

Instrukcja przygotowania lokalnego środowiska do pracy nad projektami organizacji Ostrzolka.

## 1. Wymagania wstępne
Zanim zaczniesz, upewnij się, że masz zainstalowane:
* **Git** (wersja 2.x lub nowsza)
* **Node.js** (LTS)
* **Docker** (opcjonalnie)

## 2. Klonowanie repozytoriów
Wszystkie nasze projekty znajdują się w organizacji `ostrzolka`. Aby zacząć pracę, wykonaj:

```bash
git clone [https://github.com/ostrzolka/nazwa-projektu.git](https://github.com/ostrzolka/nazwa-projektu.git)
cd nazwa-projektu
npm install
```

## 3. Kontakt w razie problemów
Jeśli napotkasz błędy podczas instalacji:
* Sprawdź zakładkę Issues w głównym repozytorium.
* Napisz na naszym kanale komunikacyjnym.
